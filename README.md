 # 基於區塊鏈技術的募款平台 

 ## 為加強捐款人信任度與捐款意願，利用區塊鏈技術之透明性，結合傳統募款網頁前端，使每一筆金流皆可在區塊鏈上查詢得到

 ## 前端使用React + daisyUI，編譯器使用Vite，後端採用MongoDB，區塊鏈建構在Polygon Mumbai測試鏈上
