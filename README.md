 ## 基於區塊鏈技術的募款平台

為提高捐款者對募款平台的信任並激發更多的捐款意願，我結合區塊鏈技術與傳統的募款網頁前端。這將確保每一筆金流都具有極高的透明度，因為捐款者可以輕鬆地在區塊鏈上查詢到相關的交易資訊

---
前端部分採用React和daisyUI，編譯工具使用Vite。後端選用MongoDB作為資料庫。而區塊鏈則搭建於Polygon Mumbai測試鏈上。
